    اليوم الآخر ( آخر يوم من الأيام )
    ****************************** **************************
    .............................. .............................. ..................

    يوم القيامة هو اليوم الآخر .. هو آخر يوم من الأيام ..
    سوف يأتى بغتة على العباد وهم لايشعرون .. سوف يأتِهم كلمح البصر أو أقرب من ذلك مما لايعِه الفكر ولا تدركه العقول ..
    قد يأتِنا هذا اليوم ونحن أحياء .. وقد يأتنا ونحن أموات ..
    ونظراً لاختلاف الوقت بين البلاد والأقطار فى الليل والنهار .. فسيأتى على بعض الأمم والأقوام فى العشية أو فى الليل وهم نائمون ..
    وعلى البعض فى الضحى أو فى النهار وهم لاهون لاعبون ..
    { أَفَأَمِنَ أَهْلُ الْقُرَى أَن يَأْتِيَهُمْ بَأْسُنَا بَيَاتاً وَهُمْ نَآئِمُونَ أَوَ أَمِنَ أَهْلُ الْقُرَى أَن يَأْتِيَهُمْ بَأْسُنَا ضُحًى وَهُمْ يَلْعَبُونَ }
    ...
    هذا هو يوم الدين .. سيبعث الله فيه كل الأولين والآخرين ..
    فمنذ أن خلق الله الأرض وما عليها فإن كل من أحياه الله بها سوف يأتى بعد الممات إلى هذا الميقات ..
    { قُلْ إِن الْأَوَّلِينَ وَالْآخِرِينَ لَمَجْمُوعُونَ إِلَى مِيقَات يَوْمٍ مَّعْلُومٍ }
    الأولون : هم كل الأقوام منذ سيدنا آدم عليه السلام وحتى مبعث رسول الله صلى الله عليه وسلم .
    والآخِرون : هم كل الأمم والشعوب منذ مبعث رسول الله صلى الله عليه وسلم وحتى يوم القيامة .
    ...
    سوف ينفخ فى الصور نفخة الفزع .. ويهبّ كل العباد ويصيبهم الرعب والفزع والهلع ..
    وسوف ينفخ فى الصور نفخة الصعق وستُصعق كل الأحياء .. وستأتى نفخة البعث ويقوم الناس لرب العالمين .
    { وَيَوْمَ يُنفَخُ فِي الصُّورِ فَفَزِعَ مَن فِي السَّمَاوَاتِ وَمَن فِي الْأَرْضِ إِلَّا مَن شَاء اللَّهُ }
    { وَنُفِخَ فِي الصُّورِ فَصَعِقَ مَن فِي السَّمَاوَاتِ وَمَن فِي الْأَرْضِ إِلَّا مَن شَاء اللَّهُ ثُمَّ نُفِخَ فِيهِ أُخْرَى فَإِذَا هُم قِيَامٌ يَنظُرُونَ}
    ...
    ستتشقق الأرض ويخرج الناس من قبورهم وأجداثهم وهم يتناسلون وكأنهم جراد منتشر ..
    سوف يخرجون مشدوهين ناظرين مادّين أعناقهم لن ترتد إليهم أطراف وأهداب عيونهم من هول هذا اليوم العظيم ..
    { مُهْطِعِينَ مُقْنِعِي رُءُوسِهِمْ لاَ يَرْتَدُّ إِلَيْهِمْ طَرْفُهُمْ وَأَفْئِدَتُهُم ْ هَوَاء }
    ...
    إنه يومٌ عسير غير يسير .. يومٌ عبوس قمطرير .. يومٌ شره مستطير ..
    ستُمحى فيه الكواكب وتنتثر .. وتطمس النجوم وتنكدر .. وستكشط السماء وتنفطر .. وسوف تتخلى الأرض عما فيها وما عليها من أحمال وأثقال ..
    سيجمع فيه الشمس والقمر .. وستنتهى القوانين والنواميس التى وضعها الله للحياة .. فلا ضياء ولا أنوار ولا تعاقب للليلٍ أو النهار ولا مسار للنجوم والأفلاك فى المدار ..
    ظلمة وظلام وإظلام .. ورجف للأرض وزلزلة .. وبث وبس ونسف للجبال .. وتسجير وتفجير للبحور .. وبعثرة للأجداث والقبور ..
    ...
    يومٌ لاتنفع فيه حسرة أو ندم .. يومٌ لا توبة فيه ولا غفران .. فلقد طويت الصحف وجفت الأقلام ..
    { وَلَيْسَتِ التَّوْبَةُ لِلَّذِينَ يَعْمَلُونَ السَّيِّئَاتِ حَتَّى إِذَا حَضَرَ أَحَدَهُمُ الْمَوْتُ قَالَ إِنِّي تُبْتُ الآنَ .. وَلاَ الَّذِينَ يَمُوتُونَ وَهُمْ كُفَّارٌ }
    ...
    يومٌ لاخفايا فيه ولا أسرار .. سوف يفتضح كل ما أخفيناه وسيُبلى أمام الله والخلائق كل ما أسررناه .
    { يَوْمَ تُبْلَى السَّرَائِرُ } .. { يَوْمَئِذٍ تُعْرَضُونَ لَا تَخْفَى مِنكُمْ خَافِيَةٌ }
    ...
    سوف ينظر المرء ماقدمت يداه .. ويتذكر ما فعل وما سعى .. وسيعض الظالم على يديه .. وسيقول الكافر يا ليتنى كنت ترابا ..
    سوف يفر المرء من أخيه ومن أمه وأبيه لا الولد سينفع والده ولا الوالد سينفع ولده ..
    سيفر المرء من زوجته وبنيه ومن فصيلته التى تؤويه ويتمنى أن يلقِى بهم فى النار وبكل من فى الأرض لينجو بنفسه من عذاب المنتقم الجبار .
    { يَوْمَ يَفِرُّ الْمَرْءُ مِنْ أَخِيهِ وَأُمِّهِ وَأَبِيهِ وَصَاحِبَتِهِ وَبَنِيهِ لِكُلِّ امْرِئٍ مِّنْهُمْ يَوْمَئِذٍ شَأْنٌ يُغْنِيهِ }
    {يَوَدُّ الْمُجْرِمُ لَوْ يَفْتَدِي مِنْ عَذَابِ يَوْمِئِذٍ بِبَنِيهِ وَصَاحِبَتِهِ وَأَخِيهِ وَفَصِيلَتِهِ الَّتِي تُؤْويهِ وَمَن فِي الْأَرْضِ جَمِيعاً ثُمَّ يُنجِيهِ}
    ...
    سوف يجازى كل إنسان عما فعل واحتقب وسيُحاسب عن كل ما كسب واكتسب ..
    وإن كان الناس قد نسوا أقوالهم وأعمالهم فالله لم ينساها وإن لم يحصوا أفعالهم فالله قد أحصاها ..
    { هَذَا كِتَابُنَا يَنطِقُ عَلَيْكُم بِالْحَقِّ .. إِنَّا كُنَّا نَسْتَنسِخُ مَا كُنتُمْ تَعْمَلُونَ }
    { وَيَقُولُونَ يَا وَيْلَتَنَا مَالِ هَذَا الْكِتَابِ لَا يُغَادِرُ صَغِيرَةً وَلَا كَبِيرَةً إِلَّا أَحْصَاهَا }
    ...
    سوف تتبدل الأرض غير الأرض والسماوات ..
    وبعد الظلمة والظلام سيأتى الإبصار وتشرق الأنوار بنور الواحد القهار .. يقول العزيز الجبار :
    { وَأَشْرَقَتِ الْأَرْضُ بِنُورِ رَبِّهَا وَوُضِعَ الْكِتَابُ وَجِيءَ بِالنَّبِيِّينَ وَالشُّهَدَاء }
    ستوضع الموازين .. ويأتى العباد كل منا طائره فى عنقه يقرأ كتابه الذى يلقاه منشورا .. سوف يقرأه متعلماً كان أو غير متعلم ..
    { اقْرَأْ كَتَابَكَ كَفَى بِنَفْسِكَ الْيَوْمَ عَلَيْكَ حَسِيباً }
    ...
    من آمن واتقى وأطاع الله ورسله .. لاخوف عليهم ولاهم يحزنون ..
    لا فزع لهم يوم ينفخ فى الصور .. ولا ظلمة لهم أو ظلام .. نورهم يسعى بين أيديهم وبأيمانهم ..
    { يَوْمَ تَرَى الْمُؤْمِنِينَ وَالْمُؤْمِنَات ِ يَسْعَى نُورُهُم بَيْنَ أَيْدِيهِمْ وَبِأَيْمَانِهِ م بُشْرَاكُمُ الْيَوْمَ }
    { لَا يَحْزُنُهُمُ الْفَزَعُ الْأَكْبَرُ وَتَتَلَقَّاهُم ُ الْمَلَائِكَةُ هَذَا يَوْمُكُمُ الَّذِي كُنتُمْ تُوعَدُونَ }
    هؤلاء هم أصحاب اليمين .. فى جنة الخلد والنعيم .. لهم مقام أمين وفى نعيم مقيم وجنات تجرى من تحتها الأنهار ..
    { لَهُم مَّا يَشَاؤُونَ فِيهَا وَلَدَيْنَا مَزِيدٌ }
    ...
    ومن كفر وعصى ولم يطع الله ورسله .. فموردهم وخيم وعذابهم أليم وحالهم ذميم .. فى سموم وحميم وظل من يحموم ..
    صبروا أو لم يصبروا فلن يخرجوا ..
    { اصْلَوْهَا فَاصْبِرُوا أَوْ لَا تَصْبِرُوا سَوَاء عَلَيْكُمْ إِنَّمَا تُجْزَوْنَ مَا كُنتُمْ تَعْمَلُونَ }
    هؤلاء هم أصحاب الشمال .. مأواهم نار جهنم وسكنى الجحيم .. لابثين فيها دهوراً وأحقاباً خالدين .. صُماً لايسمعون بكماً لاينطقون عمياً لايبصرون ..
    { وَنَحْشُرُهُمْ يَوْمَ الْقِيَامَةِ عَلَى وُجُوهِهِمْ عُمْياً وَبُكْماً وَصُمّاً مَّأْوَاهُمْ جَهَنَّمُ كُلَّمَا خَبَتْ زِدْنَاهُمْ سَعِيراً }
