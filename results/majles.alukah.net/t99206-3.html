    ووقفت على قول بعضهم: ( إسنادُ كالشمس ) و( إسْنَادٌ كَالشَّمْسِ مِنْ أَوَّلِهِ إلَى آخِرِهِ. ) و( وَهَذَا سَنَدٌ كَالشَّمْسِ فِي الصِّحَّةِ ) و( فَهَذَا إسْنَادٌ كَالشَّمْسِ لا يَسَعُ أَحَدًا خِلافُهُ. ) و( أسانيد كالشمس ) قالها ابن حزم مثلاً في المحلى وفي (5/ 11) قال: ( وَهَذَا أَثَرٌ كَالشَّمْسِ صِحَّةً. ) اهـ و في (11/ 235) قال: ( هذا إسناد صحيح كالشمس لا مغمز فيه ) اهـ وفي شرح ابن ماجه لمغلطاي (ص: 1263): ( لأنه مسند كالشمس مرية في صحته ) اهـ وفي سير أعلام النبلاء للذهبي (14/ 448): ( هذا حديث صحيح متفق عليه ، وإسناده كالشمس وضوحا. ) اهـ وهذا كله بمعنى رقم 41 فيُضاف له فهو بمعناه.
