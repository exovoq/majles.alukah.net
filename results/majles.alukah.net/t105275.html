    السلام عليكم ورحمة الله وبركاته

    تشابه في المعنى وبعض الكلمات وأرقام الآيات !!!
    ( 1 )
    قوله تعالى : في سورة القصص
     وَجَآءَ رَجُلٌ مِّنۡ أَقۡصَا ٱلۡمَدِينَةِ يَسۡعَىٰ قَالَ يَمُوسىٰٓ إِنَّ ٱلۡمَلَأَ
    يَأۡتَمِرُونَ بِكَ لِيَقۡتُلُوكَ فَٱخۡرُجۡ إِنِّي لَكَ مِنَ ٱلنَّصِحِينَ (20)
    فهذه الآية متشابهة مع آية سورة ( يس ) من قوله تعالى :
    وَجَآءَ مِنۡ أَقۡصَا ٱلۡمَدِينَةِ
    رَجُلٌ يَسۡعَىٰ قَالَ يَٰقَوۡمِ ٱتَّبِعُواْ ٱلۡمُرۡسَلِينَ (20)

    ،،،،،،،،،، ومن الآيات أيضاً ،،،،،،،،،،
    ( 2 )
     قوله تعالى : في سورة الفرقان
    وَعَادٗا وَثَمُودَاْ
    وَأَصۡحَٰبَ ٱلرَّسِّ وَقُرُونَۢا بَيۡنَ ذَٰلِكَ كَثِيرٗا (38)
    مع آية سورة ( العنكبوت ) من قوله تعالى :
    وَعَادٗا وَثَمُودَاْ وَقَد تَّبَيَّنَ لَكُم
    مِّن مَّسَٰكِنِهِمۡۖ وَزَيَّنَ لَهُمُ ٱلشَّيۡطَٰنُ أَعۡمَٰلَهُمۡ
    فَصَدَّهُمۡ عَنِ ٱلسَّبِيلِ وَكَانُواْ مُسۡتَبۡصِرِينَ (38)

    ،،،،،،،،،، ومن الآيات أيضاً ،،،،،،،،،،
    ( 3 )
    قوله تعالى : في سورة البقرة
     أُوْلَٰٓئِكَ عَلَىٰ هُدٗى مِّن رَّبِّهِمۡۖ وَأُوْلَٰٓئِكَ
    هُمُ ٱلۡمُفۡلِحُونَ (5)
    مع آية سورة ( لقمان ) من قوله تعالى :
    أُوْلَٰٓئِكَ عَلَىٰ هُدٗى مِّن رَّبِّهِمۡۖ وَأُوْلَٰٓئِكَ
    هُمُ ٱلۡمُفۡلِحُونَ (5)

    ،،،،،،،،،، ومن الآيات أيضاً ،،،،،،،،،،
    ( 4 )
    قوله تعالى : في سورة غافر
    ٱلَّذِينَ يُجَٰدِلُونَ فِيٓ ءَايَٰتِ ٱللَّهِ بِغَيۡرِ سُلۡطَٰنٍ
    أَتَىٰهُمۡۖ كَبُرَ مَقۡتًا عِندَ ٱللَّهِ وَعِندَ ٱلَّذِينَ ءَامَنُواْۚ كَذَٰلِكَ
    يَطۡبَعُ ٱللَّهُ عَلَىٰ كُلِّ قَلۡبِ مُتَكَبِّرٖ جَبَّارٖ(35)
    مع آية سورة ( الشورى ) من قوله تعالى :
    وَيَعۡلَمَ ٱلَّذِينَ يُجَٰدِلُونَ فِيٓ ءَايَٰتِنَا مَا لَهُم مِّن مَّحِيصٖ(35)

    ،،،،،،،،،، ومن الآيات أيضاً ،،،،،،،،،،
    ( 5 )
    قوله تعالى : في سورة العنكبوت
    فَأَنجَيۡنَٰهُ وَأَصۡحَٰبَ ٱلسَّفِينَةِ وَجَعَلۡنَٰهَآ ءَايَةٗ لِّلۡعَٰلَمِينَ(1 5)
    مع آية سورة ( القمر ) من قوله تعالى :
    وَلَقَد تَّرَكۡنَٰهَآ ءَايَةٗ فَهَلۡ مِن مُّدَّكِرٖ(15)

    ،،،،،،،،،، ومن الآيات أيضاً ،،،،،،،،،،
    ( 6 )
    قوله تعالى : في سورة ص
    كَذَّبَتۡ قَبۡلَهُمۡ قَوۡمُ نُوحٖ وَعَادٞ وَفِرۡعَوۡنُ ذُو ٱلۡأَوۡتَادِ (12)
    وَثَمُودُ وَقَوۡمُ لُوطٖ وَأَصۡحَٰبُ     لۡ*َٔيۡكَةِۚ أُوْلَٰٓئِكَ ٱلۡأَحۡزَابُ (13)
    إِن كُلٌّ إِلَّا كَذَّبَ ٱلرُّسُلَ     فَحَقَّ عِقَابِ (14)
    مع آية سورة ( ق ) من قوله تعالى :
    كَذَّبَتۡ قَبۡلَهُمۡ قَوۡمُ نُوحٖ وَأَصۡحَٰبُ ٱلرَّسِّ وَثَمُودُ(12)
    وَعَادٞ وَفِرۡعَوۡنُ وَإِخۡوَٰنُ     لُوطٖ (13)
    وَأَصۡحَٰبُ ٱلۡأَيۡكَةِ وَقَوۡمُ تُبَّعٖۚ كُلّٞ كَذَّبَ ٱلرُّسُلَ فَحَقَّ وَعِيدِ     (14)

    قوله تعالى : في سورة النحل
    فَإِذَا قَرَأۡتَ ٱلۡقُرۡءَانَ فَٱسۡتَعِذۡ بِٱللَّهِ مِنَ ٱلشَّيۡطَٰنِ ٱلرَّجِيمِ(98)
    مع آية سورة (المؤمنون ) من قوله تعالى :
    وَقُل رَّبِّ أَعُوذُ بِكَ مِنۡ هَمَزَٰتِ ٱلشَّيَٰطِينِ (97)
    وَأَعُوذُ بِكَ رَبِّ أَن     يَحۡضُرُونِ (98)
    ،،،،،،،،،، ومن الآيات أيضاً ،،،،،،،،،،
    قوله تعالى : في سورة الأنبياء
    وَنُوحًا إِذۡ نَادَىٰ مِن قَبۡلُ فَٱسۡتَجَبۡنَا لَهُۥ فَنَجَّيۡنَٰهُ
    وَأَهۡلَهُۥ مِنَ ٱلۡكَرۡبِ ٱلۡعَظِيمِ (76)
    مع آية سورة ( الصافات ) من قوله تعالى :
    وَنَجَّيۡنَٰهُ وَأَهۡلَهُۥ مِنَ ٱلۡكَرۡبِ ٱلۡعَظِيمِ (76)
    ،،،،،،،،،،،،،،، ومنها أيضاً ،،،،،،،،،،،،،،،
    قوله تعالى : في سورة القصص :
    فَأَخَذۡنَٰهُ وَجُنُودَهُۥ فَنَبَذۡنَٰهُمۡ
    فِي ٱلۡيَمِّۖ فَٱنظُرۡ كَيۡفَ كَانَ عَٰقِبَةُ ٱلظَّٰلِمِينَ (40)
    مع آية سورة ( الذاريات ) من قوله تعالى :
    فَأَخَذۡنَٰهُ وَجُنُودَهُۥ فَنَبَذۡنَٰهُمۡ فِي ٱلۡيَمِّ وَهُوَ مُلِيمٞ (40)
    ،،،،،،،،،،،،،،، ومنها أيضاً ،،،،،،،،،،،،،،،
    قوله تعالى : في سورة الأعراف :
    قَالَ ٱخۡرُجۡ مِنۡهَا     مَذۡءُومٗا مَّدۡحُورٗاۖ لَّمَن تَبِعَكَ مِنۡهُمۡ
    لَأَمۡلَأَنَّ جَهَنَّمَ مِنكُمۡ أَجۡمَعِينَ (18)
     مع آية سورة ( الإسراء ) من قوله تعالى :
    مَّن كَانَ يُرِيدُ ٱلۡعَاجِلَةَ عَجَّلۡنَا لَهُۥ فِيهَا مَا نَشَآءُ لِمَن نُّرِيدُ ثُمَّ
    جَعَلۡنَا لَهُۥ جَهَنَّمَ يَصۡلَىٰهَا     مَذۡمُومٗا مَّدۡحُورٗا (18)
     ،،،،،،،،،،،،،،، ومنها أيضاً ،،،،،،،،،،،،،،،
    قوله تعالى في سورة البقرة :
    وَأَقِيمُواْ ٱلصَّلَوٰةَ وَءَاتُواْ ٱلزَّكَوٰةَ وَٱرۡكَعُواْ مَعَ ٱلرَّٰكِعِينَ (43)
    مع آية سورة ( آل عمران) من قوله تعالى :
    يَٰمَرۡيَمُ ٱقۡنُتِي لِرَبِّكِ وَٱسۡجُدِي وَٱرۡكَعِي مَعَ ٱلرَّٰكِعِينَ (43)
