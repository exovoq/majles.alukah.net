    يوم القيامة ( ستطوى الصحف وتجف الأقلام )

    يوم القيامة
    ****************************** ****************************** ****************************** ******
    .......
    يوم القيامة يومٌ مهول . يومٌ عبوسٌ قمطرير . يومٌ شره مستطير . يومٌ عسير على الكافرين غير يسير ..
    يوم القيامة ستنتهى فيه كل القوانين والنواميس التى وضعها الله للحياة ..
    سيجمع فيه الشمس والقمر ولن يكون هناك ضياء ولا نور أو أنوار . ولن يكون فيه تعاقب لليلٍ أو نهار . بل ظلمة وظلام وإظلام ..
    { يَسْأَلُ أَيَّانَ يَوْمُ الْقِيَامَةِ فَإِذَا بَرِقَ الْبَصَرُ وَخَسَفَ الْقَمَرُ وَجُمِعَ الشَّمْسُ وَالْقَمَرُ }

    سوف تتشقق السماء وتُكشط وتنفطر . وستُمحى الكواكب وتتساقط وتنتثر . وتُطمس النجوم وتتهاوى وتنكدر ..
    سوف تتخلى الأرض عما فيها وما عليها من أحمال وأثقال .. بعثرة للأجداث والقبور .
    وتسجير وتفجير للبحور . ودكٌ ورجفٌ للأرض وزلزلة .. وبث وتسيير ونسف للجبال .. سوف تتبدل الأرض غير الأرض والسماوات ..
    { يَوْمَ تُبَدَّلُ الأَرْضُ غَيْرَ الأَرْضِ وَالسَّمَاوَاتُ وَبَرَزُواْ للّهِ الْوَاحِدِ الْقَهَّارِ }
    .......
    هول يوم القيامة هولٌ عظيم لا يمكن وصفه أو بيانه فى أى مقال إلا بما أخبرنا به ذو الجلال .. فقال :

    { يَوْماً عَبُوساً قَمْطَرِيراً } . { يَوْماً كَانَ شَرُّهُ مُسْتَطِيراً } . { يَوْمٌ عَسِيرٌ عَلَى الْكَافِرِينَ غَيْرُ يَسِيرٍ }
    .......
    يوم القيامة سيأتى على العباد بغتة وهم لايشعرون . سيأتى كلمح البصر أو أقرب من ذلك مما لا تدركه أو تعِيَه وتتخيله العقول ..
    { وَمَا أَمْرُ السَّاعَةِ إِلاَّ كَلَمْحِ الْبَصَرِ أَوْ هُوَ أَقْرَبُ }
    قد يأتنا ونحن أحياء . وقد يأتنا ونحن أموات . وقد يأتنا ضحىً أو نهاراً . أو فى العشية أو الليل . وقد يأتنا نحن نائمون . أو ونحن لاهون لاعبون
    ( نظراً لاختلاف الساعات والأوقات فى الليل والنهار بين مختلف البلاد والبلدان والأقطار ) ..
    { أَفَأَمِنَ أَهْلُ الْقُرَى أَن يَأْتِيَهُمْ بَأْسُنَا بَيَاتاً وَهُمْ نَآئِمُونَ } . { أَوَ أَمِنَ أَهْلُ الْقُرَى أَن يَأْتِيَهُمْ بَأْسُنَا ضُحًى وَهُمْ يَلْعَبُونَ }
    .......
    يوم القيامة هو يوم الدين . هو اليوم الآخر . هو آخر يوم من الأيام .. يوم القيامة هو يوم الفصل بين الناس والخلق أجمعين ..
    سيبعث الله فيه كل الأولين والآخرين ..
    فمنذ أن خلق الله أبينا آدم عليه السلام فإن كل من خلقه الله وأحياه على هذه الأرض سوف يبعثه الله بعد الممات إلى هذا الميعاد والميقات ..
    { قُلْ إِن الْأَوَّلِينَ وَالْآخِرِينَ لَمَجْمُوعُونَ إِلَى مِيقَات يَوْمٍ مَّعْلُومٍ } . { هَذَا يَوْمُ الْفَصْلِ جَمَعْنَاكُمْ وَالْأَوَّلِينَ }
    .......
    سينفخ فى الصور نفخة الفزع . ويصيب الناس رعب وذعر وهلع .. وسينفخ فى الصور نفخة الصعق وستُصعق كل الأحياء ..
    وسينفخ فى الصور نفخة البعث ويقوم الناس لرب العالمين ..
    سوف تتشقق الأرض عنهم ويهبّون من قبورهم ويخرجون من أجداثهم وهم يتناسلون كأنهم جراد منتشر ..
    سيخرجون مشدوهين ناظرين مادين أعناقهم ورؤوسهم لا ترتد إليهم أهداب وأطراف عيونهم من هول هذا اليوم العبوس القمطرير ..
    { وَيَوْمَ يُنفَخُ فِي الصُّورِ فَفَزِعَ مَن فِي السَّمَاوَاتِ وَمَن فِي الْأَرْضِ إِلَّا مَن شَاء اللَّهُ }
    { وَنُفِخَ فِي الصُّورِ فَصَعِقَ مَن فِي السَّمَاوَاتِ وَمَن فِي الْأَرْضِ إِلَّا مَن شَاء اللَّهُ . ثُمَّ نُفِخَ فِيهِ أُخْرَى فَإِذَا هُم قِيَامٌ يَنظُرُونَ }
    { مُهْطِعِينَ مُقْنِعِي رُءُوسِهِمْ لاَ يَرْتَدُّ إِلَيْهِمْ طَرْفُهُمْ وَأَفْئِدَتُهُم ْ هَوَاء }
    .......
    يوم القيامة يومٌ لاخفايا فيه ولا أسرار . سوف يفتضح فيه كل ما أخفيناه . وسيُبلى أمام الله والخلائق كل الخبايا وكل ما أسررناه ..
    { يَوْمَ تُبْلَى السَّرَائِرُ } . { يَوْمَئِذٍ تُعْرَضُونَ لَا تَخْفَى مِنكُمْ خَافِيَةٌ }
    لا الوالد سينفع ولده ولا الإبن سينفع والده . سيفر المرء فيه من أخيه . وسيفر من أمه وأبيه . ومن زوجته وبنيه . سيتمنى أن يلقِى بهم جميعاً فى النار
    وبكل من فى الأرض لينجو بنفسه من عذاب الواحد القهار والمنتقم العزيز الجبار ..
    { يَوَدُّ الْمُجْرِمُ لَوْ يَفْتَدِي مِنْ عَذَابِ يَوْمِئِذٍ بِبَنِيهِ وَصَاحِبَتِهِ وَأَخِيهِ وَفَصِيلَتِهِ الَّتِي تُؤْويهِ وَمَن فِي الْأَرْضِ جَمِيعاً ثُمَّ يُنجِيهِ }
    .......
    يوم القيامة هو يوم الحسرة والندامة . لاضراعة فيه ولاشفاعة ولاحجة فيه ولابرهان ولاناصر فيه أو نصير ..
    يوم القيامة لا توبة فيه ولا غفران .. ستطوى فيه الصحف وتجف الأقلام ..
    { وَلَيْسَتِ التَّوْبَةُ لِلَّذِينَ يَعْمَلُونَ السَّيِّئَاتِ حَتَّى إِذَا حَضَرَ أَحَدَهُمُ الْمَوْتُ قَالَ إِنِّي تُبْتُ الآنَ . وَلاَ الَّذِينَ يَمُوتُونَ وَهُمْ كُفَّارٌ }
    { وَاتَّقُواْ يَوْماً لاَّ تَجْزِي نَفْسٌ عَن نَّفْسٍ شَيْئاً وَلاَ يُقْبَلُ مِنْهَا عَدْلٌ وَلاَ تَنفَعُهَا شَفَاعَةٌ وَلاَ هُمْ يُنصَرُونَ }
    .......
    يوم القيامة سوف يتذكر الإنسان فيه ما سعى . وسينظر المرء ما قدمت يداه فى هذه الحياة .
    سوف يتذكر الإنسان ما كان منه وما اكتسبه واحتقبه وكل ما قاله وما فعله .. سوف يعض الظالم على يديه . ويقول الكافر يا ليتنى كنت ترابا ..
    يوم القيامة سيأتى كل منا وطائره فى عنقه ليقرأ كتابه الذى يلقاه منشورا .. ( سيقرأه متعلماً كان أو غير متعلم ) ..
    { اقْرَأْ كَتَابَكَ كَفَى بِنَفْسِكَ الْيَوْمَ عَلَيْكَ حَسِيباً }
    { وَيَقُولُونَ يَا وَيْلَتَنَا مَالِ هَذَا الْكِتَابِ لَا يُغَادِرُ صَغِيرَةً وَلَا كَبِيرَةً إِلَّا أَحْصَاهَا }
    .......
    ستوضع الموازين .. وبعد الظلمة والظلام سيأتى الإبصار وتشرق الأنوار بنور الواحد القهار .. يقول العزيز الجبار :
    { وَأَشْرَقَتِ الْأَرْضُ بِنُورِ رَبِّهَا وَوُضِعَ الْكِتَابُ وَجِيءَ بِالنَّبِيِّينَ وَالشُّهَدَاء }
    سيسأل الله الأنبياء والمرسلين . وستُسأل كل الأمم والأقوام ..
    من عصى الله ولم يطع نبيه ورسوله فهو من المجرمين الفجار .. ومن آمن بالله وأطاع نبيه ورسوله فهو من الأتقياء الأبرار ..
    { لَنَسْأَلَنَّ الَّذِينَ أُرْسِلَ إِلَيْهِمْ وَلَنَسْأَلَنَّ الْمُرْسَلِينَ }
    .......
    سوف يجازى كل إنسان عما فعل واحتقب .. وسيُحاسب عن كل ما كسب واكتسب . وإن كان الناس قد نسوا أقوالهم فالله لم ينساها
    وإن لم يحصوا أفعالهم فالله قد أحصاها .. سوف تشهد عليها أيديهم وأرجلهم وستنطق بها ألسنتهم وجوارحهم ..
    { هَذَا كِتَابُنَا يَنطِقُ عَلَيْكُم بِالْحَقِّ إِنَّا كُنَّا نَسْتَنسِخُ مَا كُنتُمْ تَعْمَلُونَ }
    { يَوْمَ تَشْهَدُ عَلَيْهِمْ أَلْسِنَتُهُمْ وَأَيْدِيهِمْ وَأَرْجُلُهُم بِمَا كَانُوا يَعْمَلُونَ }
    .......
    وبعد السؤال والحساب . سيكون المآل والمآب .
    إما أن يكون الإنسان من الذين ابيضت وجوههم أو ممن اسودت وقترت وغبرت وجوههم . إما أن يكون من السعداء أو ممن كتب عليهم الشقاء ..
    إما أن يكون من أصحاب اليمين أو من أصحاب الشمال .. فما بعد الدنيا من دار إلا الجنة أو النار ..
    { فَرِيقٌ فِي الْجَنَّةِ وَفَرِيقٌ فِي السَّعِيرِ }
    .......
    من آمن وأطاع ( من الأولين والآخرين ) .. فقد كفلهم الله برحمته واستثناهم بمشيئته ووقاهم من شر هذا اليوم العسير وحفظهم الله من هوله المستطير ..
    لا فزع ولاخوف لهم ولا ظلمة أو ظلام .. نورهم يسعى بين أيديهم وبأيمانهم ..
    { فَوَقَاهُمُ اللَّهُ شَرَّ ذَلِكَ الْيَوْمِ } .
    { لَا يَحْزُنُهُمُ الْفَزَعُ الْأَكْبَرُ وَتَتَلَقَّاهُم ُ الْمَلَائِكَةُ هَذَا يَوْمُكُمُ الَّذِي كُنتُمْ تُوعَدُونَ }
    { يَوْمَ تَرَى الْمُؤْمِنِينَ وَالْمُؤْمِنَات ِ يَسْعَى نُورُهُم بَيْنَ أَيْدِيهِمْ وَبِأَيْمَانِهِ م بُشْرَاكُمُ الْيَوْمَ }
    هؤلاء هم أصحاب اليمين ..
    جزاؤهم جنة الخلد والنعيم . فى نعيم مقيم لهم مقام أمين ولهم جنات تجرى من تحتها الأنهار ..
    { وَالَّذِينَ آمَنُواْ وَعَمِلُواْ الصَّالِحَاتِ أُولَـئِكَ أَصْحَابُ الْجَنَّةِ هُمْ فِيهَا خَالِدُونَ } .
    { لَهُمْ فِيهَا مَا يَشَاؤُونَ خَالِدِينَ }
    .......
    ومن كفر وعصى ( من الأولين والآخرين ) .. فموردهم وخيم وحالهم ذميم وعذابهم أليم فى سموم وحميم وظل من يحموم ..
    { لَا بُشْرَى يَوْمَئِذٍ لِّلْمُجْرِمِين َ } . { وَمَن يَعْصِ اللَّهَ وَرَسُولَهُ فَإِنَّ لَهُ نَارَ جَهَنَّمَ خَالِدِينَ فِيهَا أَبَداً }
    هؤلاء هم أصحاب الشمال .. مأواهم سكنى الجحيم وفى نار جهنم لابثين ماكثين فيها دهوراً وأحقاباً خالدين .. صُماً لايسمعون
    بكماً لا ينطقون عمياً لايبصرون .. صبروا أو لم يصبروا : لن يخرجوا ..
    { وَنَحْشُرُهُمْ يَوْمَ الْقِيَامَةِ عَلَى وُجُوهِهِمْ عُمْياً وَبُكْماً وَصُمّاً مَّأْوَاهُمْ جَهَنَّمُ كُلَّمَا خَبَتْ زِدْنَاهُمْ سَعِيراً }
    { اصْلَوْهَا فَاصْبِرُوا أَوْ لَا تَصْبِرُوا سَوَاء عَلَيْكُمْ .. إِنَّمَا تُجْزَوْنَ مَا كُنتُمْ تَعْمَلُونَ }
